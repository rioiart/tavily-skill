# Tavily Researcher

Web research via Tavily's LLM-optimized APIs: search, extract, sitemap, and research.

## Setup

Requires `TAVILY_API_KEY` environment variable. Get a free key (1000 credits/month) at https://app.tavily.com

## Choosing the Right Tool

| Need | Script |
|------|--------|
| Web search results | `tavily_search.py` |
| Content from specific URLs | `tavily_extract.py` |
| Discover URLs on a site (fast) | `tavily_sitemap.py` |
| AI-synthesized report with citations | `tavily_research.py` |

## Quick Start

### Search
```bash
uv run scripts/tavily_search.py "What is retrieval augmented generation?" --include-answer
```

### News Search
```bash
uv run scripts/tavily_search.py "AI regulation updates" --topic news --time-range week
```

### Extract Content from URLs
```bash
uv run scripts/tavily_extract.py https://example.com/article1 https://example.com/article2
```

### Discover URLs on a Site
```bash
uv run scripts/tavily_sitemap.py https://docs.example.com --max-depth 2 --limit 100
```

### AI Research Report
```bash
uv run scripts/tavily_research.py "Compare LangGraph vs CrewAI for multi-agent systems" --model pro
```

## Scripts

### `tavily_search.py`
Standard web search. Returns titles, URLs, snippets, and optional AI answer.

**Key options:**
- `--depth ultra-fast|fast|basic|advanced` ‚Äî Trade speed for relevance
- `--topic general|news|finance` ‚Äî Optimize for content type
- `--include-answer` ‚Äî Get AI-generated summary
- `--include-raw-content` ‚Äî Get full page content
- `--time-range day|week|month|year` ‚Äî Filter by recency
- `--chunks-per-source N` ‚Äî Chunks per source (1-5, advanced/fast)
- `--include-domains` / `--exclude-domains` ‚Äî Filter sources

### `tavily_extract.py`
Extract full content from specific URLs (up to 20 at once).

**Key options:**
- `--depth basic|advanced` ‚Äî Use advanced for JS-heavy pages
- `--query TEXT` ‚Äî Rerank chunks by relevance
- `--chunks-per-source N` ‚Äî Return only relevant chunks (1-5, requires --query)
- `--format markdown|text` ‚Äî Output format

### `tavily_sitemap.py`
Discover URLs on a website without extracting content. Faster than crawl ‚Äî use to understand site structure first.

**Key options:**
- `--max-depth N` ‚Äî Crawl depth 1-5 (start with 1)
- `--limit N` ‚Äî Total URLs cap (default: 50)
- `--instructions TEXT` ‚Äî Focus on specific types of pages

### `tavily_research.py`
AI-synthesized research with citations. Takes 30-120 seconds.

**Key options:**
- `--model mini|pro|auto` ‚Äî mini for focused queries, pro for comprehensive analysis
- `--citation-format numbered|mla|apa|chicago`
- `--output-schema JSON` ‚Äî Get structured JSON output
- `--output-file PATH` ‚Äî Save results to file

## Cost Optimization

| Task | Recommended Approach | Credits |
|------|---------------------|---------|
| Quick fact check | `tavily_search.py --include-answer` | 1 |
| Deeper search | `tavily_search.py --depth advanced` | 2 |
| Full article content | `tavily_extract.py <urls>` | 1 per 5 URLs |
| Discover site URLs | `tavily_sitemap.py` | 1 |
| AI research report | `tavily_research.py` | varies |

## Domain-Specific Search
```bash
# Only search specific sites
uv run scripts/tavily_search.py "async python" --include-domains docs.python.org,realpython.com

# Exclude unreliable sources
uv run scripts/tavily_search.py "health advice" --exclude-domains reddit.com,quora.com
```
